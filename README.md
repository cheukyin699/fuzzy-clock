# fuzzy-clock

![Python 2.7, 3.6](https://img.shields.io/badge/python-2.7,%203.6-blue.svg)

A clock that displays time in plain English.

## Features

Prints the time in plain English. Nothing else.

## Usage

Can be used as a script in i3blocks.

```sh
python flock.py
```
